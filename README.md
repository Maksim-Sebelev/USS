# USS (Usefull-Shell-Scripts)

В данном репозитории предоставлено несколько shell-скриптов, которые могут немного облегчить вам жизнь.

# Установка:
```bash
git clone https://github.com/Maksim-Sebelev/USS.git
cd USS
```

Чтобы установить скрипты:
```bash
sudo bash install [имена нужных скриптов из папки src]
```
Пример уставновки `cmt` и `tall`:
```bash
sudo bash install cmt tall
```

`cmake-deps` - создает картинку с зависимостями библиотек, которые генерирует cmake. Принимает 2 аргумента - путь до папки с CMakeLists.txt и путь до папки build. Если аргументов нет - то первый = '../', а второй = './' (мы в папке build). выходной файл будет создан в `build/dot-out` и называется `dependencies.dot`.


`tall` - (touch all) применяет `touch` ко всем файлам в текующей директории. аргументы не принимает. Кому надо, тот знает :).

`cmt` - (cmake target) заменяет любую систему сборки одной командой. Больше не надо помнить под какую систему сборки вы собрали ваш проект, для сборки нужного таргета просто вызовите `cmt <target>`.
